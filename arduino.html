Arduino Project: Pac-Man Scroller

Course: Computer Science Principles
Date: May 2019


This Arduino project was challenging, but worthwhile because it gave me a chance to work with hardware which I've wanted to do for a while now. I liked working with the C language and seeing what I can do with it. For the project, my partner Jake and I designed a "game" that is a 3 by 3 matrix of sprites on the LCD that forms a big Pac-Man figure that scrolls across the screen and then proceeds to loop back to the beginning of the screen when holding a button down. (There are unused open mouth sprites too but we did not have enough time to fully animate it properly) The rightmost column on the LCD counts Pac-Man's speed on the top row (0 at rest and 1 when moving) and the number of completed screen loops on the bottom row. Before the LCD is initialized, in a reference to the way the original Pac-Man arcade machine works, the short Pac-Man intro jingle plays on a piezo buzzer, which when finished displays the game as it should look. It was a lot of trouble to get the button to work because for some reason it would not register until I used a new pin and reorganized all my code, and then I had to come up with a way to clear the spaces behind Pac-Man's sprite, which I solved by creating an empty sprite to replace the passed up sections of screen.
